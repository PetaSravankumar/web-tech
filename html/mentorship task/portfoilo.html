Projects

Consumer Law GPT – AI-Powered Legal Chatbot

Consumer Law GPT is a multilingual AI-driven legal assistant designed to help users understand and exercise their consumer rights under the Indian Consumer Protection Act. It explains legal policies in simple language and provides step-by-step guidance for consumer complaints, refunds, product/service issues, and consumer court procedures. The chatbot supports multiple Indian languages, making legal information more inclusive and accessible for people from different regions. It also maintains session-based conversation context, enabling personalized responses tailored to the user’s query history.

Tech Stack: React, TypeScript, FastAPI (Python), Node.js, MongoDB, Groq LLM API, Hugging Face

Real-Time Weather Forecasting Web App

This project is a responsive web application that provides real-time weather data for any city across the globe using the OpenWeather API. It displays key weather details including current temperature, humidity, wind speed, visibility, and overall climate conditions through a clean and intuitive interface. The application offers instant city-based search results and includes error handling for invalid inputs or API issues to ensure a smooth user experience. Its mobile-friendly UI ensures seamless usage across devices, making it useful for travel planning and daily weather checks.

Tech Stack: HTML, CSS, JavaScript, OpenWeather API
